## 1. 功能概述

本文档描述战斗场景中以下三个新增流程的设计规格：

| 流程 | 触发条件 | 简述 |
|------|---------|------|
| 英雄登场 | 玩家进入战斗（每 Wave 开始）| 英雄从场外飞入战场，顺序落地 |
| 切队 | 第 1 支队伍全灭 | 第 2 队以登场动画形式进入战场，显示"再战！！！"提示 |
| WAVE 提示 | 玩家切换队伍后，战斗正式开始 | 屏幕显示"波数：X/X"横幅动画 |

## 2. UI新增：玩家队伍标识

### 2.1 功能说明

在战斗界面顶部新增「队伍图标」，直观展示玩家当前携带的队伍数量及状态。

### 2.2 图标状态规则

| 场景 | 图标表现 |
|------|---------|
| 无第 2 支队伍 | 不显示队伍 icon |
| 有 2 支队伍（均存活）| 显示 2 个 icon，颜色正常 |
| 第 1 队有英雄阵亡 | 第 1 队 icon 部分变灰（死几个灰几个） |
| 第 2 队有英雄阵亡 | 第 2 队 icon 部分变灰（死几个灰几个） |
| 某支队伍全灭 | 该支队伍所有 icon 全部变灰 |

> **注意**：  
> - icon 数量对应该队伍实际英雄数量（非固定 3 个）  
> - 当玩家队伍全败（场上所有英雄死亡），icon 立即全部变灰  

### 2.3 位置

战斗界面顶部 HUD 区域，紧邻当前已有的「波数」显示右侧（见图 1）。

## 3. UI新增：战斗开始标识（WAVE 提示）

### 3.1 功能说明

每当一场新的战斗波次正式开始时，屏幕显示"波数：X/X"横幅提示，参考现有 WAVE 提示样式。

### 3.2 触发时机

- **Wave 开始**：玩家首次进入战斗，在英雄落地完毕、三消棋盘激活之前播放  
- **切队后重新开战**：第 1 队全灭后，第 2 队完成登场动画结束时触发，出现"波数"提示后进入正式战斗  

### 3.3 UI 样式

- 金色/橙色横幅，居中显示  
- 文字内容：`波数：X / X`（与顶部 HUD 波数一致）  
- 播放完毕后自动消失，棋盘解锁可操作

## 4. 流程一：战斗开始 / 英雄登场

### 4.1 流程序列

```
[进入战斗]
    ↓
[1] 战场加载画面（黑屏/转场）
    ↓
[2] 英雄登场 · 第一阶段
    → 第 1 名英雄飞入，落地，播放飞入动画
    ↓
[3] 英雄登场 · 第二阶段
    → 其余英雄依次或同时飞入，各自落地到对应站位
    ↓
[4] 前排关卡敌方出现
    → 敌方展示：英雄、波次信息同时出现
    ↓
[5] 棋盘入场
    → 三消棋盘从下方滑入，英雄画面缩小至上方战斗区
    ↓
[6] 使物触发提示
    → 普通音效+BGM 起，提示玩家开始操作（BOSS入场情况特殊，单独处理）
```

### 4.2 各步骤详细说明

#### [1] 战场加载画面

- 进入战斗时播放短暂黑屏/转场特效
- 同时加载关卡资源、英雄资源

#### [2][3] 英雄登场动画

| 参数 | 规格 |
|------|------|
| 入场方式 | 英雄从场景上方飞入，带飞行轨迹特效（蓝色光效，见图示） |
| 落地顺序 | 第 1 名英雄先飞入落地 → 其余英雄飞入落地 |
| 站位 | 按照阵容队列的站位规则对齐（近/远排） |
| 落地动作 | 落地时播放落地震动/尘土小特效 |
| 入场时长 | 建议总时长不超过 2.5 秒（可跳过） |
| 可跳过 | 是（点击屏幕任意区域跳过，直接到达棋盘就绪状态）|

#### [4] 敌方出现

- 敌方在英雄全部落地后出现
- 显示敌方名称、血条、波次数（参考已有关卡逻辑）

#### [5] 棋盘入场

- 棋盘从底部滑入
- 棋盘入场与「顶部 HUD 显示完整」同步完成
- 棋盘静止后，操作解锁

#### [6] 使物触发提示

- BGM 完整启动
- 显示 WAVE 提示横幅（见第 3 节）

## 5. 流程二：英雄死亡 / 切队

### 5.1 触发条件

第 1 支队伍所有英雄全部阵亡时，进入切队流程（前提：玩家携带了第 2 支队伍）。

### 5.2 流程序列

```
[第 1 队全灭]
    ↓
[1] 英雄死亡 · 第 1 队全灭表现
    → 第 1 队英雄模型灰化/消失（不打断当前操作，等消除动作完成后进入死亡表现）
    → 顶部队伍 icon 全部变灰
    ↓
[2] 队伍选择画面（前线）
    → 战场上方切换到空场景
    → 棋盘变为不可操作（不可继续消除，要求来不及死亡的也停止）
    ↓
[3] 第 2 队英雄登场（动画）
    → 英雄依次飞入，落地，同入场流程（参考第 4 节）
    ↓
[4] 确认更新/激活战斗
    → 第 2 队落地完毕
    → 顶部 icon 更新为第 2 队状态
    → 显示"再战！！！"文字提示（橙色横幅居中）
    ↓
[5] 重新开始/继续战斗
    → "再战！！！"消失
    → 显示 WAVE 提示横幅（波数：X/X）
    → 棋盘重新解锁，可操作
```

### 5.3 各步骤详细说明

#### [1] 第 1 队全灭表现

| 表现项 | 规格 |
|------|------|
| 触发时机 | 最后一名第 1 队英雄血量归零时 |
| 英雄表现 | 死亡动画播放（倒地 or 消散），随后模型消失 |
| 棋盘状态 | 如棋盘正在消除中，等当前消除链结束再触发切队流程 |
| 顶部 icon | 第 1 队所有 icon 瞬间变灰 |
| 音效 | 播放全队阵亡音效（待定） |

#### [2] 队伍切换过渡

- 战场上方区域清空（第 1 队英雄消失后）
- 棋盘锁定（变暗/不可点击），提示切换进行中
- 过渡时间 < 0.5 秒，视觉上平滑衔接

> **注意**：切队过渡中**不可进行**任何消除操作；如英雄正在执行技能动画，等动画完成后进入死亡表现，不可在技能打断时死亡。

#### [3] 第 2 队登场动画

- 同「战斗开始 - 英雄登场」流程一致（可共用动画规格）
- 第 2 队英雄按照阵容站位飞入落地
- 落地时光效颜色可与第 1 队区分（如颜色变暖 or 更亮的白光）

#### [4]"再战！！！"提示

| 属性 | 规格 |
|------|------|
| 文字 | 再战！！！ |
| 样式 | 橙色/金色加粗字体，带描边 |
| 动画 | 缩放弹出 + 停顿 0.8s + 淡出 |
| 位置 | 屏幕正中 |
| 音效 | 振奋音效（类战鼓/号角） |

#### [5] 棋盘重新激活

- "再战"消失后，显示 WAVE 横幅（波数：X/X）
- WAVE 横幅消失后，棋盘解锁，可操作

## 6. 美术需求拆解

### 6.1 UI 资源

| 序号 | 资源名称 | 规格要求 | 备注 |
|------|---------|---------|------|
| UI-01 | 队伍 icon（激活态） | 2D 图标，圆形头像框格式 | 与英雄头像关联，每英雄一个 |
| UI-02 | 队伍 icon（灰化态） | UI-01 的灰度版本 | 可由程序置灰，美术提供原图 |
| UI-03 | WAVE 提示横幅 | 九宫格可拉伸，金橙色风格 | 带描边光效 |
| UI-04 | "再战！！！"文字贴图 | 完整字图或字体+特效分离 | 加粗字体 + 橙色描边，适配多语言（需英/日/韩版） |
| UI-05 | 棋盘锁定遮罩 | 半透明深色蒙层 | 切队过渡中棋盘不可交互时显示 |

### 6.2 角色动画资源

| 序号 | 资源名称 | 动画类型 | 备注 |
|------|---------|---------|------|
| ANIM-01 | 英雄飞入动画 | 骨骼/帧动画 | 从场景顶部飞入落地，带光效拖尾 |
| ANIM-02 | 英雄落地效果 | 特效动画 | 落地时地面震动/尘土粒子 |
| ANIM-03 | 英雄死亡动画 | 骨骼/帧动画 | 全队阵亡专属死亡动作（区别于单兵死亡）|
| ANIM-04 | 英雄灰化/消散特效 | 粒子特效 | 死亡后模型消失的过渡特效 |

### 6.3 特效资源

| 序号 | 资源名称 | 特效类型 | 备注 |
|------|---------|---------|------|
| FX-01 | 英雄登场光柱 | 粒子/Spine 特效 | 英雄落地前的蓝色光柱（如图示），第 2 队可用不同颜色 |
| FX-02 | 落地冲击波 | 粒子特效 | 英雄落地瞬间，地面小范围扩散特效 |
| FX-03 | "再战"出场特效 | Spine/Shader | 文字弹出时的金光特效 |
| FX-04 | WAVE 横幅出场特效 | Spine/UI 动画 | 横幅飞入屏幕中央的动画 |

### 6.4 音效资源

| 序号 | 资源名称 | 类型 | 备注 |
|------|---------|------|------|
| SFX-01 | 英雄入场音效 | 音效 | 飞入+落地，节奏感强 |
| SFX-02 | 全队阵亡音效 | 音效 | 沉重、失败感 |
| SFX-03 | "再战"音效 | 音效 | 振奋感，战鼓/号角风格 |
| SFX-04 | WAVE 提示音效 | 音效 | 提示性短音，与现有 WAVE 风格一致 |

### 6.5 美术优先级建议

| 优先级 | 内容 | 理由 |
|--------|------|------|
| P0 | ANIM-01 英雄飞入动画、FX-01 登场光柱 | 核心流程视觉主体，其他功能依赖此动画 |
| P0 | UI-01/02 队伍 icon | 影响战斗 HUD 完整性 |
| P1 | UI-04 "再战"文字、FX-03 出场特效 | 切队流程的视觉高潮点 |
| P1 | ANIM-03/04 死亡+消散 | 流程完整性 |
| P2 | SFX-01 ~ 04 全部音效 | 视觉可先行，音效可后配 |
| P2 | UI-05 棋盘锁定遮罩 | 程序可先用纯色临时方案 |

## 7. 技术备注

| 项目 | 说明 |
|------|------|
| 动画可跳过 | 英雄登场动画需支持点击跳过，程序需实现状态机跳转 |
| 切队状态机 | 需要新增 `TeamSwitch` 状态，与现有战斗状态机对接 |
| Icon 灰化 | 建议由程序对原图 icon 做灰度处理，无需美术单独出灰化图（节省资源包） |
| 多语言 | "再战！！！"与 WAVE 提示文字需纳入 i18n，避免写死图片 |
| 棋盘锁定 | 切队过渡中棋盘 `InputEnabled = false`，并叠加视觉遮罩 |
| 跳过边界 | 跳过动画时，需直接跳到"棋盘就绪"状态，英雄直接在目标站位生成，不出现中间帧 |
