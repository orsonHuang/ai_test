# 项目：Awakening Demo

## 项目类型
独立游戏Demo — AI对话探索解谜，30分钟单局流程

## 用户原始需求
- 跟电脑AI对话探索文件的游戏
- AI初始懵懂，开机后可让AI查看电脑文件
- 包含密码解锁、不断交互聊天探索
- 最后发现电脑主人背后的秘密
- 混合模式：90%规则模板 + 10%AI生成（控制运营成本）
- 框架先行，文本后续迭代

## 工作约定
- 文本内容用占位符 `[FILL:简短描述]` 标注，方便用户后续填
- 修改GDD时同步更新 08-iteration-log.md
- 代码改动同步更新 GDD/07-tech.md
- 每次会话开始先读 GDD/08-iteration-log.md 了解最新进展
- 用户的游戏设计想法改 GDD 优先，不要直接动代码

## 目录约定
- GDD/ = 设计文档（用户主导）
- knowledge/ = 知识库（用户主导的剧情内容）
- engine/ = 代码实现（AI主导）
- tests/ = 测试用例
