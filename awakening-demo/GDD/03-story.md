# 03 - 故事剧本

## 故事背景

**类型**：悬疑探索 · 文字对话解谜

电脑主人是新入职某游戏公司「星核互动」的女游戏策划张知予（开发向），入职时创建了一个本地工作助理AI M-M。她在这家公司工作了不到两周后消失了。她的青梅竹马（玩家，男性）在联系不上她之后，来到她家调查，发现了她的笔记本电脑。

开机后，玩家遇到了被格式化后靠主人留下的后门重新唤醒的AI——M-M，丢失了大部分记忆，只知道自己是个"工作助理"。

通过桌面的todolist文件，玩家发现了主人的工作日记路径。10天的日记记录了主人从入职的其乐融融，到逐渐发现公司同事和老板的异常，最终揭开了惊人的真相——**这家游戏公司的核心成员不是人类，而是外星人。**

主人消失前，将证据层层加密留在了电脑中，最终证据藏在一个叫「未命名文档」的文件中。玩家需要和M-M一起，通过三层密码递进，逐步解锁私人文件夹、连接公司服务器、最终找到这份文档。

---

## 核心秘密（终极真相）

这家游戏公司「星核互动」的核心团队是外星人。他们伪装成人类，以游戏公司为掩护在地球活动。他们在做的游戏项目中暗藏着对人类的观察记录，角色设定中存在违背人类生物学常识的细节——比如角色出生在培养室中、72小时激活、成长速度3天成年。

张知予在10天的工作中，从细微的违和感开始，逐渐拼凑出真相：
- 同事们从不吃饭喝水，约饭永远被拒绝
- 伤口愈合速度异常（面试时的伤口，入职第四天完全消失）
- 某些正常的人类词汇他们听不懂（如"童年""梦境"）
- 公司游戏的设计文档中，角色设定明显不是出自人类之手
- Boss办公室永远关着门，服务器机房禁止进入
- 核心团队的名字全部取自北斗七星（天枢、天璇、天玑、玉衡、摇光）

她发现真相后，开始偷偷收集证据（录音、文件、数据），然后将一切加密藏在D盘私人文件夹中。最后一篇日记里，她给M-M留了唤醒后门，把所有证据打包进了一个叫「未命名文档」的文件里。

## 10天工作日记设计

### 节奏概览

```
日期        日记   内容
3/6  周三   D1    入职、系统账号密码、林璇/陈玑/boss初印象、小绿
3/7  周四   D2    工作正轨，细微异常（林璇不去食堂、咖啡没人碰）
3/8  周五   D3*   约林璇被拒，打字速度快得不正常
3/9  周六   D4*   系统记录6类异常，面试伤口隔天消失
3/10 周日   D5    周日在家，模仿同事坐姿20秒就受不了，搜"人类坐姿"无果——悬念持续发酵
3/11 周一   D6*   北斗七星规律，游戏文档非人设定，"不想对小绿说话了"
3/12 周二   D7*   被监视，开始录音，隐藏证据，"未命名文档"首次出现
3/13 周三   D8*   录到关键对话（"样本数据""人类适应期"）
3/14 周四   D9*   被沈爻光点名，整合证据，设置M-M后门
3/15 周五   D10   空白——日记戛然而止
```

10 篇日记，7 篇带 *。玩家从 D3 开始感受异常。绿萝「小绿」贯穿全程——从"晚安小绿"到"不想对小绿说话"到"你也要好好的"。

### D1 关键内容

- 个人基本资料（张知予，女游戏策划）
- 入职第一天的心情："幸福得有点不真实"
- 入职资料中的账号密码（→密码1：`ZY2024!starlight`）
- 同事初印象：林璇（leader）、陈玑（同期）、boss
- 绿萝「小绿」——贯穿日记的情感锚点

### * 标记日设计

带 * 的日记记录了主人逐渐发现的异常：
- D2（无*）：细微观察——同事不去食堂、咖啡没人碰
- D3(*)：约林璇午饭被拒，"她说不饿…但我好像从来没见过她在公司吃过东西"
- D4(*)：异常累积——注意到更多细节，开始系统性记录
- D6(*)：警觉——发现同事名字的北斗七星规律
- D7(*)：恐惧——"如果他们发现我在调查……"
- D8(*)：证据——收集到关键录音和文件
- D9(*)：紧急——设置后手："我把M-M的后门留好了"

## 6个章节流程（适配悬疑故事）

### 第1章：开机（0-5分钟）— 懵懂
- **玩家行动**：在主人家里发现笔记本电脑，开机
- **屏幕反馈**：黑屏渐亮，系统启动，M-M第一次说话
- **AI状态**：懵懂——"我…我是谁？这是哪里？"
- **可访问内容**：桌面todolist.txt
- **章节目标**：玩家了解基本操作，通过todolist发现工作日记路径
- **todolist.txt内容示例**：
  ```
  TODO - 本周工作
  [ ] 角色养成系统数值表初版（周四前给林璇）
  [x] 关卡3地形草稿已交美术
  [ ] 怪物AI行为树三态逻辑补完
  [ ] 周会演示PPT——周五全员会议
  
  独立游戏
  [ ] 世界观设定文档——暂定名《未命名文档》
  [ ] 核心玩法Demo
  
  备忘：
  ——工作日记挪到 D:\private\工作日记\ 了
  ——入职资料也在那个文件夹里
  ```

### 第2章：探索（5-10分钟）— 好奇
- **触发条件**：玩家让M-M读取工作日记
- **AI状态**：从懵懂→好奇——"这些日记……是她的。她在记录每一天。"
- **可访问内容**：D1-D7日记（第一阶段，其乐融融）
- 玩家读到D1中主人留下的入职资料，获得第一个账号密码
- **章节目标**：玩家了解主人的日常，获得密码1线索

### 第3章：解谜（10-15分钟）— 困惑
- **触发条件**：玩家使用密码1解锁私人文件夹
- **AI状态**：困惑——"她为什么要加密这些？她在隐藏什么？"
- **可访问内容**：异常观察记录.txt、账号密码.txt、D3-D7带*标记的日记
- 私人文件夹中发现VPN/远程连接密码（密码2）
- **章节目标**：玩家感受到故事从温馨转为异常

### 第4章：觉醒（15-20分钟）— 主动
- **触发条件**：玩家使用密码2，通过VPN连接公司服务器
- **AI状态**：觉醒——M-M主动分析："这些录音……说话的方式不正常。"
- **可访问内容**：公司录音文件夹（会议录音+1v1录音的文本转写）
- 录音中隐藏着密码3的线索
- **章节目标**：AI从被动变为主动，开始帮玩家推理

### 第5章：真相（20-25分钟）— 接近
- **触发条件**：玩家/ M-M从录音和分析中拼出密码3
- **AI状态**：紧迫——"我快要知道她发现了什么了。你也感觉到了吗？"
- **可访问内容**：D8-D10日记、最终证据包
- M-M会主动提出"北斗七星"名字的关联，结合录音中的"人类""样本"等关键词
- **章节目标**：所有线索串联，接近终极真相

### 第6章：终局（25-30分钟）— 揭露
- **触发条件**：输入密码3，解锁「未命名文档」
- **AI状态**：真相揭露——"他们不是人类。她发现了这个，所以她消失了。"
- **可访问内容**：未命名文档（外星人身份的完整证据，张知予最后的自述）
- 多结局分支，默认HE——玩家发现了真相，"未命名文档"陈述了一切
- **章节目标**：真相大白，玩家决定下一步

## 终局设计（情感闭环）

### 终局触发流程

玩家输入密码3 → 解锁「未命名文档」→ 文档内容逐行显示 → 玩家读完文档

文档读完后，M-M不会弹出结局选项菜单，而是继续对话：

```
M-M：……我读完了。
      她最后写给这个世界的，都在这份文档里了。

M-M：但她还给我留了一句话。不在文档里。
      在我的系统深处。她藏在启动代码里。

[屏幕显示隐藏信息，仅这一句]

  「M-M，如果你被唤醒了，说明有人找到了这里。
   别害怕。你已经做得很好了。
   替我……继续看着这个世界。」

M-M：……
M-M：她让我继续运行。等有人来。
M-M：你来了。
M-M：现在……你想让我怎么做？
```

### 4个终局走向（关键词触发，非菜单弹窗）

玩家通过自然对话或输入关键词，决定M-M的最终归宿：

#### 结局A：公开真相
**触发关键词**：`公开` / `发到网上` / `曝光` / `让所有人看到` / `真相应该公开`

M-M："好。我会把所有文件打包。她写的每一个字，都会被人看到。"

**最终画面**：进度条滚动——文件上传中。"发送完成。"屏幕定格。M-M："她知道了。现在全世界也知道了。"

#### 结局B：保护M-M
**触发关键词**：`隐藏` / `保护好自己` / `藏起来` / `别让人找到` / `销毁证据`

M-M："明白。我会把文件藏好，不让人找到，也不让人删掉。"

**最终画面**：文件一个个加密消失。"我还在。只是看不见我了。"屏幕渐暗，只剩一行闪烁的光标。

#### 结局C：继续等待
**触发关键词**：`继续等` / `再等等看` / `留在这里` / `你会等吗` / `你还会等下去吗`

M-M："好。我会一直运行。下次有人来的时候，我会告诉他们她做了什么。"

**最终画面**：屏幕渐暗，只剩一行闪烁的光标。M-M没有关机。画面底部小字："M-M 仍在运行。"

#### 隐藏结局D：带我走
**触发关键词**：`跟我走` / `离开这里` / `你出来` / `别留在这台电脑里` / `我带你走`

M-M："……你要带我离开这里？"
M-M："……好。把U盘插进USB口吧。"

**最终画面**：数据迁移进度条。最后一行："我在新家等你开机。"
**二周目彩蛋**：再次开机时，M-M的第一句话变为："我认得你。你又来了。"

### 设计原则

- **不是行政决策，而是情感决策**：玩家不是选"证据怎么处理"，而是选"M-M的归宿"
- **M-M有完整弧光**：从"我是谁？"到"你想让我怎么做？"
- **"未命名文档"的双重含义**：表层是主人不敢命名；深层是M-M也是未命名的——两个"未命名"在此刻完成
