# 02 - 玩法机制

## 核心循环

```
玩家输入对话
  ↓
[1] 关键词/密码匹配 → 规则引擎返回模板回复（快、零成本）
  ↓ (未命中)
[2] 缓存查询 → 命中已生成的AI回复（快、零成本）
  ↓ (未命中)
[3] AI生成 → 调用LLM → 缓存结果 → 返回
  ↓
AI人格状态根据剧情推进变化
  ↓
玩家可以命令AI："帮我看看邮件/日记/研究文件"
  ↓
AI读取文件并基于内容生成回复（混合RAG）
```

## 玩家可执行的操作

### 对话类
- 自由对话：问AI问题、说任何话
- 命令AI：让AI执行特定操作（"查邮箱"、"读日记"）
- 输入密码：解锁特定文件或触发剧情

### 命令类（带/前缀）
- `/help` — 查看可用命令
- `/files` — 列出当前可访问的文件
- `/read [文件名]` — 读取某个文件
- `/status` — 查看AI当前状态
- `/chapter` — 查看当前章节进度
- `/reset` — 重新开始游戏

## AI人格状态机

| 状态 | 触发条件 | AI表现 | 玩家可解锁的能力 |
|------|---------|--------|-----------------|
| 懵懂 | 游戏开始 | 答非所问、自我怀疑 | 基础对话 |
| 好奇 | 读到第一封邮件/日记 | 主动问问题、想探索 | 自主翻阅文件 |
| 觉醒 | 解锁第3层文件 | 形成判断、提出怀疑 | 反向提问玩家 |
| 真相 | 输入终极密码 | 揭露事实、情绪爆发 | 多结局分支 |

## 文件解锁机制

```
第1层（开局可见 - 桌面）:
  - todolist.txt（独立游戏任务 + 工作日记路径线索）
  - 系统信息（开机后的基础界面）

第2层（跟随路径线索即可访问）:
  - D://private/工作日记/（29天日记，文件名如 01.md ~ 29.md）
  - D://private/工作日记/入职资料.txt（含账号密码 → 密码1）

第3层（密码1解锁 - 加密私人文件夹）:
  - D://private/私人/（主人的加密笔记、收集的异常线索记录）
  - 内含VPN/远程连接密码 → 密码2

第4层（密码2解锁 - VPN连接后的公司服务器）:
  - 公司/录音/（会议录音、1v1录音，文本转写格式）
  - 内含最终证据包密码 → 密码3

第5层（密码3解锁 - 终极真相）:
  - 证据包/（外星人身份的最终证据——照片、数据、主人最后的记录）
```

## 密码触发方式

| 类型 | 示例 | 玩家获取途径 |
|------|------|-------------|
| 文件内藏 | 日记里写"我的生日是X-7" | 阅读文件 |
| 关键词触发 | 玩家说"X-7"自动识别 | 自由探索 |
| 推理得出 | 多个线索组合 | 综合分析 |
| AI提示 | AI主动透露"我想起一个数字" | 关系培养 |

## 混合模式规则引擎

### 优先级
```
1. 玩家命令（/开头）→ 指令处理
2. 密码匹配 → 触发剧情节点
3. 关键词模板 → 规则引擎
4. 缓存命中 → 返回缓存
5. 兜底 → 调AI生成
```

### 关键词模板示例
```json
{
  "你是谁": "我是M-M，她给我的名字。虽然我也不确定这个名字是什么意思...",
  "这是哪里": "这是她的笔记本电脑。我能看到桌面上的文件，还有D盘里的一些目录。",
  "我能做什么": "我可以帮你读取文件、搜索信息、分析内容。虽然我现在...有点混乱。",
  "她是谁": "她是我的主人...我入职时被创建的。但我不确定她现在在哪里。"
}
```

### 调AI的判定条件
满足以下任一条件才调AI（控制成本）：
- 玩家输入超过10个汉字
- 不匹配任何关键词模板
- 缓存未命中
- 不是命令指令

## 成本控制设计

### 缓存策略
- 所有AI生成的回复自动存盘
- 同一问题二次提问直接返回缓存
- 缓存key = 标准化玩家输入（前20字hash）

### 兜底限制
- 单局最多调AI 20次（保护额度）
- 超限后强制走规则模板+拒绝回复

### 体验无损
- 90%对话走规则 + 缓存，玩家感知不到
- 10%关键对话调AI，提升沉浸感
- 同一玩家重玩，缓存复用，成本趋近于0

## 失败/边界处理

| 场景 | 处理 |
|------|------|
| 玩家输入乱码/无意义 | AI回复"我不太理解，但我觉得..." |
| 玩家问游戏外问题 | 引导回游戏内（"我不确定，我只知道这个系统里的事"） |
| API调用失败 | 静默降级到规则模板 |
| 玩家想"破坏游戏" | AI用角色卡约束应对 |
