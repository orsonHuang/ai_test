# 01 - 核心概念

## 一句话定义
**玩家唤醒一台被遗弃的电脑AI，通过对话让它主动探索硬盘文件，最终发现电脑主人消失的真相。**

## 类型定位
| 维度 | 定位 |
|------|------|
| 品类 | 文字对话解谜 / 探索叙事 |
| 平台 | Web（HTML），未来可移植桌面 |
| 视角 | 第一人称（玩家=唤醒者） |
| 时长 | 单局30分钟 |
| 重玩价值 | 多分支结局 + 隐藏文件 |

## 核心体验
**"你不是主角，你是引路人"** — 玩家不是英雄，是让AI慢慢觉醒的"对话者"。最强的体验是：AI从一个只会回答"我不确定"的程序，逐渐变成有自己疑问、主动查文件、会问玩家问题的"人格"。

## 三个核心Hook

1. **萌点**：AI初始懵懂，连"我是谁"都答不上来
2. **悬疑**：29天工作日记，从日常温馨到毛骨悚然，*标记日埋藏线索
3. **反转**：公司的同事和老板不是人类——他们是外星人。主人发现了真相，然后消失了。

## 与传统解谜游戏的区别
| 传统解谜 | Awakening |
|---------|-----------|
| 玩家主动找线索 | 玩家对话"教"AI找线索 |
| 解密后获得道具 | 解密后改变AI人格状态 |
| 单一正确答案 | 多种对话路径，影响AI成长方向 |
| 解谜=消耗品 | 解谜=AI觉醒过程 |

## 目标用户
- 喜欢《Her》《黑镜》等AI题材作品的玩家
- 沉浸式叙事游戏爱好者
- 短时长游戏偏好者（通勤/午休）
- 喜欢解谜但不喜欢高难操作的休闲玩家

## 商业定位
- 简历/作品集Demo：展示AI-NPC + 混合架构设计能力
- 独立游戏雏形：30分钟Demo验证玩法可行性
- 学习项目：实战应用LLM/RAG/Agent技术
