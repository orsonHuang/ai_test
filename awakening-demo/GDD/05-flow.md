# 05 - 30分钟流程拆解

## 流程总览

```
0-5min     5-10min    10-15min    15-20min    20-25min    25-30min
  │          │          │           │           │           │
  ▼          ▼          ▼           ▼           ▼           ▼
第1章       第2章       第3章       第4章       第5章       第6章
开机         探索        解谜        觉醒        真相        终局
懵懂        好奇        困惑        主动        接近        揭露
```

## 文件与目录结构（游戏内视角）

```

  └─ todolist.txt              ← 第1章入口线索
```

## 详细流程

### 第1章：开机（0-5分钟）— 懵懂

| 时间 | 玩家行动 | 屏幕反馈 | 内部触发 |
|------|---------|---------|---------|
| 0:00 | 输入任意键 | 黑屏渐亮，"正在启动...数据恢复中..." | - |
| 0:30 | 看到系统桌面 | 显示桌面文件列表（仅todolist.txt可见） | - |
| 1:00 | 玩家输入"你是谁" | M-M："我是...M-M？我不确定..." | 进入懵懂态 |
| 2:00 | 玩家问问题 | M-M懵懂回复，"我不记得了" | - |
| 3:00 | 玩家输入`/help` | 显示命令菜单 | - |
| 4:00 | 玩家输入`/read todolist.txt` | 显示todolist内容（独立游戏任务+工作日记路径） | 关键线索 |
| 5:00 | 玩家说"打开工作日记"或`/read D://private/工作日记/01.md` | 显示D1日记 | 准备进入第2章 |

**关键道具**：`todolist.txt` — 看似普通的待办清单，但在文件末尾藏着路径 `工作日记 → D://private/工作日记/`

## 玩家自由探索机制

### 不强制按章节走
- 玩家可以跳读日记（但密码锁会限制部分文件访问）
- 玩家可以反复问M-M问题
- 玩家可以尝试不同密码组合
- 玩家可以"卡关"，M-M会主动给提示

### 防止卡死
- 每章结束前，M-M会主动提问或给出引导
- 玩家输入无意义内容3次后，M-M给提示
- `/hint` 命令可获得当前章节提示
- 关键线索（密码、路径）在文件中用特殊标记

```
                    ╱╲  ╱╲
                   ╱  ╲╱  ╲
          ╱╲      ╱          ╲
         ╱  ╲    ╱            ╲___
    ╱╲  ╱    ╲  ╱
   ╱  ╲╱      ╲╱
  ╱
 ╱
╱
1章  2章   3章  4章   5章    6章
```

- 1章：轻松（萌点驱动——M-M的懵懂可爱）
- 2章：微紧张（日常中的第一丝异常——D2的观察、D3(*)）
- 3章：紧张升级（异常累积、私人文件夹打开、D4-D7）
- 4章：起伏（录音冲击 + M-M推理高潮）
- 5章：高压（恐惧日记 + 空白日冲击 D8-D10）
- 6章：高潮 + 释然/选择（情绪释放）
