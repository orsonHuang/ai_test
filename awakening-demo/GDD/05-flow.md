# 05 - 30分钟流程拆解

## 流程总览

```
0-5min     5-10min    10-15min    15-20min    20-25min    25-30min
  │          │          │           │           │           │
  ▼          ▼          ▼           ▼           ▼           ▼
第1章       第2章       第3章       第4章       第5章       第6章
开机         探索        解谜        觉醒        真相        终局
懵懂        好奇        困惑        主动        接近        揭露
```

## 玩家自由探索机制

### 不强制按章节走
- 玩家可以跳读文件
- 玩家可以反复问AI问题
- 玩家可以尝试不同密码
- 玩家可以"卡关"，系统会AI主动给提示

### 防止卡死
- 每章结束前，AI会主动提问引导
- 玩家输入无意义内容3次后，AI给提示
- `/hint` 命令可获得当前章节提示

## 体验节奏设计

### 紧张感曲线
```
        ╱╲
       ╱  ╲           ╱╲
      ╱    ╲    ╱╲   ╱  ╲
     ╱      ╲  ╱  ╲ ╱    ╲
    ╱        ╲╱    ╲      ╲___
___╱
1章  2章   3章  4章  5章    6章
```
- 1-2章：轻松（萌点驱动）
- 3章：紧张（解谜驱动）
- 4章：起伏（AI变化驱动）
- 5章：紧张升级（真相接近）
- 6章：高潮 + 释然（情绪释放）
