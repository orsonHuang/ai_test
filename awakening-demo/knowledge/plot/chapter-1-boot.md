# 章节1：开机 - 懵懂

## 状态
AI状态：懵懂（dormant）
玩家可见文件：welcome.txt, readme.txt

## 关键对话

```
玩家：你是谁？
AI（懵懂）：我是... 我不太确定。让我想一想。
           等等，我想起来了。
           好像是 [FILL:AI的名字]？但我不确定这是我的还是别人的。

玩家：这是谁的电脑？
AI（懵懂）：我不确定。我能看到一些文件，但大部分是加密的。
          有些文件名我能读懂，有些像是被故意打乱的。

玩家：你能做什么？
AI（懵懂）：我...不确定。你能告诉我吗？
```

## 章节目标
让玩家理解这是对话型游戏，AI是懵懂的、不知道自己是谁的。
