## 一、他们不是人类

以下结论基于为期9天的系统性观察、录音取证和交叉验证。

### 1. 生理证据

- **林璇**：入职面试（约3月1日）左手手背有伤口，3月9日复查完全消失，间隔不足一周，无疤痕。正常人类伤口愈合周期7-14天且留疤。
- **陈玑**：每日午餐固定A套餐，从未更换，进食过程无表情变化，无对食物味道评价。像"知道要吃东西"但"不享受食物"。
- **全员**：茶水间咖啡除我之外无人取用。午休时段办公室几乎无进食行为发生。12人会议室坐姿极度一致，无翘腿、无转笔、无看手机。

### 2. 命名规律

核心团队五人姓名与北斗七星高度对应：

| 姓名 | 对应星 | 位置 |
|------|--------|------|
| 陆天枢 | 天枢 | 北斗一 |
| 林璇 | 天璇 | 北斗二 |
| 陈玑 | 天玑 | 北斗三 |
| 何恒 | 玉衡 | 北斗五 |
| 姚光 | 摇光 | 北斗七 |

五人涵盖五颗星，按恒星序号排列。非随机命名。

### 3. 语言证据

- 沈爻光在3月8日全员会议中说："**人类**的创造力是最有效的资源。"——使用"人类"而非"员工"，疑似非人类自我指代。
- 陆天枢在3月13日录音中说："**人类**的适应期比我们预估的长。"——"我们"和"人类"在同一句子中。语法不会说谎。

### 4. 游戏设定文档

项目文档中反复出现以下设定（非正常科幻创作思路）：

1. 角色初始状态为"休眠体"
2. "培养室"中72小时激活，成长期仅3天
3. 成年体直入战斗序列，无需教育和社会化过程
4. 角色无需进食、无需睡眠
5. "对某些人类基础概念无理解——如梦境、童年、孤独感"

这些设定细节过于具体，缺乏正常虚构的"留白"。更像是：在描述某种真实存在的生物特性。

## 五、张知予的自述

我3月6日入职星核互动。

9天后，我发现了一件事——我所在的这家公司，核心团队可能不是人类。

我不知道他们是什么。不知道他们从哪里来。不知道"起源计划"的完整内容。

而我唯一的武器，是一个叫M-M的AI助理，和一本还没写完的日记。

我把所有证据都藏在这个文档里了。
