## 关键技术决策

| 决策项 | 选择 | 理由 |
|--------|------|------|
| 架构 | 混合模式（90%规则+10%AI） | 控制运营成本 |
| 后端 | Python + Flask | 用户熟悉，单文件易部署 |
| 前端 | 原生HTML+JS | 无构建步骤，单文件部署 |
| AI API | 百炼Coding Plan | 用户已有Key |
| 部署 | 腾讯云Ubuntu + Nginx | 用户已有服务器 |
| 数据 | JSON文件 | 无需数据库 |
