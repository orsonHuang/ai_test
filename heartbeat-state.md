## 输出结束时 · 自主判断

> 本段由 heartbeat-state.md 自己承载判断逻辑——不依赖 AI 记住。

1. 读「未 commit 改动」字段
2. **若 = 是**：
   - 本轮有文件写入/修改？→ 收尾条件成立
   - 用户本消息含「收尾 / 收工 / 完成 / 提交」？→ 收尾意图确认
   - 收尾流程是否已在本轮跑过？（看下方「wrap-up 完成」字段）
   - **若需要收尾且未跑过** → 立即执行收尾三件套（更新 memory + NOW + 追加 observations），全部完成后最后一步 commit
   - **若已跑过** → 跳过
3. **若 = 否** → 无需动作
