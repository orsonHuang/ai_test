### 设计内容

1. **回合制 RPG 战斗原型**：4 人小队（剑士/学者/神官/舞娘）vs 3 波敌人（狼/暗法师/精英魔将）
2. **BP 增幅系统**：每回合积累 BP，消耗 BP 可增幅普攻（多段连击）或技能（倍率提升）
3. **弱点破防机制**：6 种弱点类型（斩/突/打/火/冰/光），命中弱点扣盾，盾归零 → 眩晕 + 受伤加深
4. **调试面板**：实时状态查看 + 6 项可调参数热调 + 一键重置 + 战斗日志

### 名词解释

| **名词** | **说明** |
|---|---|
| BP | Boost Points，增幅点数。每回合 +1（上限 5），用于强化普攻或技能 |
| 盾值 | 敌人的防御层数，只有命中弱点类型才能削减 |
| 破防 (Break) | 敌人盾值归零后的状态：眩晕 1 回合 + 受伤倍率 1.5x |
| 行动条 | 按 SPD 排序的单位出手顺序，显示在屏幕顶部 |
| 蓄力技 | 精英魔将特有机制：预告 3 回合后释放 2.0x 高伤，破防可打断 |

# 设计决策理由

| 决策点 | 选择了什么 | 排除了什么 | 理由 | 对实现的约束 |
|--------|-----------|-----------|------|-------------|
| 空间模型 | 队列对排（无网格） | 网格战棋 | 聚焦技能选择而非位置博弈，降低 UI 复杂度 | 无需实现移动/寻路系统 |
| 回合流转 | 行动条按 SPD 排序 | 固定轮次交替 | 让"速度"维度有策略价值（舞娘拉条才有效） | 需要环形队列管理 |
| BP 普攻机制 | 多段连击（每段独立破防） | 单段增幅 | 让普攻 BP 消耗成为快速破防手段，与技能 BP 增幅形成差异化 | 需要 for 循环 + 每段独立 emit 事件 |
| 渲染策略 | 全量 renderAll（每事件触发） | 增量 DOM 更新 | 回合制频率低（~1-2s/次），全量刷新简单可靠 | 转实时战斗时需重构为 Canvas/虚拟DOM |

# 系统功能

## 战斗系统

### 行动顺序与回合

1. **行动队构建**：所有存活单位按 SPD 降序排列，构成环形队列
2. **回合开始处理**：
   - 玩家角色：BP += BP_RATE（上限 5），SP += SP_REGEN（上限 maxSp），buff 回合 -1，清除防御状态
   - 敌人：破防倒计时检查（到期恢复），buff 回合 -1
3. **行动执行**：队首单位行动 → 重新排入队尾
4. **中断条件**：全灭敌人（进入下一波/胜利）或全灭己方（失败）

### BP 增幅系统

1. **积累规则**：每个角色回合开始时 BP += BP_RATE（默认 1），上限 5
2. **消耗方式**：
   - 普攻：每消耗 1 BP，额外打出 1 段攻击（每段独立判定破防 + 伤害计算）
   - 技能：每消耗 1 BP，伤害倍率 ×1.5（1BP）/ ×2.0（2BP）/ ×3.0（3BP）
3. **不消耗**：选择防御时 BP 不消耗，仍正常累积

### 弱点破防机制

1. **弱点类型**（6 种）：斩、突、打、火、冰、光
2. **扣盾规则**：攻击类型匹配敌人弱点 → 盾值 -1；不匹配 → 不扣盾
3. **破防状态**：盾值归零 → 敌人眩晕 1 回合 + 受伤倍率 = BREAK_DMG_BONUS（默认 1.5×）+ 打断蓄力
4. **恢复规则**：破防持续到该敌人下一次回合开始，恢复后盾值重置为满

### 伤害计算

```
baseDamage = ATK × skillMultiplier × bpMultiplier × breakBonus - DEF × 0.3
finalDamage = max(1, floor(baseDamage)) × DAMAGE_MULT
```

- ATK/DEF 受 buff 影响（如 ATK↑30%、DEF↑30%）
- breakBonus = 1.5（破防时）或 1.0（正常时）
- 防御姿态时受伤减半（×0.5）

### 角色与技能

| 角色 | 普攻 | 技能 1 | 技能 2 |
|------|------|--------|--------|
| 剑士（HP800/SP60） | 斩 | 横斩：斩·全体·0.7×·12SP | 蓄力斩：斩·单体·2.5×·18SP·固定耗 2BP |
| 学者（HP500/SP150） | 打 | 火球术：火·单体·1.8×·10SP | 冰霜风暴：冰·全体·1.2×·22SP |
| 神官（HP650/SP120） | 打 | 治愈术：回复 ATK×0.8·8SP | 圣光：光·单体·1.6×·14SP + 自回 20% |
| 舞娘（HP550/SP100） | 突 | 月光之舞：全体 ATK↑30%·2回合·15SP | 暗影步：突·单体·1.4×·10SP + 拉队友前移 2 位 |

### 敌人与波次

| 波次 | 敌人 | HP | 弱点 | 盾值 | 特殊行为 |
|------|------|-----|------|------|---------|
| 1 | 狼×2 + 暗法师×1 | 300/400 | 斩 / 突·光 | 2/3 | 暗法师自身 DEF↑ |
| 2 | 狼×2 + 暗法师×1 + 精英×1 | 300/400/900 | 同上 + 打·冰 | 2/3/4 | +精英蓄力技 |
| 3 | 狼×1 + 暗法师×2 + 精英×1 | 同上 | 同上 | 同上 | 同上 |

### AI 行为（按优先级）

1. 若被破防 → 跳过回合
2. 精英蓄力就绪 → 释放 2.0× 高伤（目标 HP 最低角色）
3. 精英可蓄力 → 开始蓄力（倒计时 3）+ 自身 DEF↑
4. 暗法师无 DEF↑buff → 自身 DEF↑30%·3回合
5. 攻击 HP 最低角色

### 调试面板

- **左侧**：实时状态（阶段/波次/当前行动者/回合计数/存活单位/各角色 HP）+ 战斗日志（上限 50 条）
- **右侧**：6 个可调参数滑块（DAMAGE_MULT / ENEMY_HP_MULT / BP_RATE / SHIELD_MULT / SP_REGEN / BREAK_DMG_BONUS）
- **一键重置**：回到初始状态，不刷新页面
