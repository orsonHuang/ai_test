# 即时三消RPG · 原型目录

> **这是什么**：即时制多英雄三消 RPG 战斗原型——时间持续流逝中通过消除匹配驱动 5 位英雄的普攻/重击/绝招。
> **开发阶段**：V1 · 待试用验证
> **创建时间**：2026-06-10
> **来源**：通过 FromDemoToSpec「开始原型验证」指令生成

---

## 项目简介

7×6 棋盘即时三消 + 5 英雄 vs 多波敌人的 RPG 战斗。核心验证目标：玩家在持续的时间压力和血量变化下，能否在"优先消哪个颜色（决定哪个英雄攻击）"和"攒能量出绝招还是持续输出"之间做出有意识的策略分化？最担心的翻车点：时间压力压倒一切，玩家最终只看棋盘不看战斗区。
