# 挂机装备RPG · 原型目录

> **这是什么**：挂机增量装备管理原型——宝箱自动产出→自动开箱→装备择优替换→金币升级→闯关验证的闭环体验。
> **开发阶段**：V6 · 已交付下游
> **创建时间**：2026-06-10
> **来源**：通过 FromDemoToSpec「开始原型验证」指令生成

---

## 项目简介

挂机增量放置类装备游戏原型（参照寻道大千/疯狂骑士团风格）。核心验证"产出-替换-升级-闯关"闭环中玩家能否持续感受成长爽快感，且不需要复杂决策就能保持投入。经过 6 个主版本迭代（v1→v6）以及 v6 试玩反馈打磨期 5 波微调 + 1 次失败回退（战力公式重写废弃），原型验证通过。附带完整的 Spec 文档 + 策划文档沉淀。

1. 先读 `交付说明.md` 了解以哪个文件为准和实现范围

3. 以 `挂机装备RPG-v6_spec.md` + `挂机装备RPG-v6_spec-technical.md` 为实现源

5. 查看 `CHANGELOG.md` 了解完整 v1→v6 迭代历史和决策链路
